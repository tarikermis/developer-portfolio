# Project Brief: Developer Portfolio

## Project Overview

A modern, responsive developer portfolio website built with Next.js that showcases Tarik Ermis's professional experience, skills, and projects as a Full Stack Developer and Blockchain Expert.

## Core Requirements

- **Personal Branding**: Professional portfolio showcasing expertise in full-stack development, blockchain, and mobile applications
- **Responsive Design**: Mobile-first approach with Bootstrap integration
- **Performance**: Optimized loading with dynamic imports and Next.js features
- **SEO Optimization**: Comprehensive meta tags and structured data
- **GitHub Integration**: Dynamic GitHub profile data fetching
- **Interactive Elements**: Lottie animations and smooth scrolling effects

## Key Sections

1. **Hero/Greetings**: Personal introduction with animated elements
2. **Skills**: Technical skills with visual progress bars and icons
3. **Experience**: Professional work history with company logos
4. **Education**: Academic background
5. **Projects**: Portfolio of completed work with live links
6. **Testimonials**: Client feedback and recommendations
7. **GitHub Profile**: Dynamic GitHub statistics integration

## Target Audience

- Potential employers and clients
- Fellow developers and collaborators
- Business partners and stakeholders

## Success Metrics

- Fast loading times (< 3s)
- High SEO scores (90%+)
- Mobile responsiveness across all devices
- Professional presentation of skills and experience
- Easy navigation and user experience

## Technical Constraints

- Must be deployable on Vercel/Netlify
- SEO-friendly with proper meta tags
- Accessible design following WCAG guidelines
- Cross-browser compatibility
- Optimized images and assets
